# AI-Based-Data-Driven-Analytics-For-Predictive-Maintenance-of-Medical-Equipment

AI-based system that predicts equipment failure in advance using machine learning, helping hospitals perform proactive maintenance and reduce downtime.

🚀 Features
Predicts days-to-failure using Random Forest
Risk levels: Healthy | Warning | Critical
REST API for real-time prediction (/predict)
Secure backend with JWT authentication
Simple frontend for monitoring and data entry
🧠 Model Performance
MAE: 4.28 days
R² Score: 0.387
Trained on 3,000+ equipment records
🏗️ Tech Stack
Backend: FastAPI, SQLAlchemy, MySQL
Frontend: HTML, CSS, JavaScript
ML: Scikit-learn, Pandas, NumPy
